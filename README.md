# 有店
有店：可迭代式电商后端系统。


# 迭代版本
## v1.0
v1.0 的版本目前所有的接口设计已经开发完成。

当前版本只支持单机单应用，只支持部署在一台服务器上运行。

## v2.0
v2.0 的版本目前正在开发中。。。

目标是将原有的单应用系统做成分布式系统。
